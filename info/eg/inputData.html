Inputs

Inputs include the samples, capture kit, reference genome build, variant annotation sources that will be used in VEP (such as dbSNP, CADD, ClinVar, etc), and Docker images.

The input data is put in a predefined format called a 'struct', which is a dictionary-like object in Python. This struct contains the following information: 

struct SampleDetails {
    String sampleId
    String sex
    Boolean proband
    String status
    Array[String] flowcellId
    Array[Array[String]] fastqs
}
