Run Workflow

This task calls the EG and tREX workflows. An if-else loop is used to choose the correct workflow based on the sample types. Specifically, the tasks that can be invoked are called runEgWorkflow and runTrexWorkflow.

These two tasks use the workflow template JSON and pipeline version tag generated by the compileWorkflow task earlier in the pipeline to download the correct workflow to DNAnexus. Once downloaded, this workflow is run on DNAnexus using the sample or family JSON files that were created by the setupBatch or setupFamilies tasks.

The workflows are run in parallel for each sample/family, and the outputs are stored in the data/ subdirectories on DNAnexus.
