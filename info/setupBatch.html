Set up Batch/Set up Families

Batches or families are set up based on the type of sequencing data (tREX/exome/genome), and prepared for analysis. Because exomes and genomes are run in trios, we handle these differently to tREX samples, which are run individually.

The task is run in an if-else loop:

  • tREX samples: Processed individually using setupBatch task.
  • Exome and genome samples: Processed in trios (maternal, paternal, and proband samples) using setupFamilies task. Families are then scattered (i.e., each individual sample within the family is processed in parallel).

The task produces a JSON per sample/family, containing each sample's FASTQ files and sample metadata (e.g., sequencing strategy, relationship status). It also sets up sample/family folders, which will be found in the /data/ subdirectories on DNAnexus.
