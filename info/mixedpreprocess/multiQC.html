MultiQC

MultiQC is a tool that aggregates results from various quality control sources (our individual FastQC results, demultiplexing statistics, and BCL Convert metrics) into a single comprehensive report for an entire batch. It generates an interactive HTML report along with a zip archive containing the underlying data.

The MultiQC task is executed within an if-else loop based on the output of the checkProject task. If the project is a genome, the wgsMultiQC task is run; otherwise, the exomeMultiQC task is executed. The primary difference between these tasks lies in the exome capture kit (retrieved from the checkProject task) and the configuration files used.
